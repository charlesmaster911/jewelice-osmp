새 콘텐츠 만들기

아래 양식을 채우면 Claude Code(또는 claude.ai)에 붙여넣을 프롬프트가 자동 생성됩니다. Claude 결과(JSON)를 받아서 "카드로 추가"하면 즉시 카피·페이스트 가능한 카드가 생성됩니다.

📌 Claude Code로 진행하는 법:
① 양식 채우기 → ② 프롬프트 카피 → ③ Claude Code 또는 claude.ai에서 새 대화 열기 → ④ 카피한 프롬프트 붙여넣기 → ⑤ Claude가 반환한 JSON 전체 카피 → ⑥ 아래 결과 영역에 붙여넣기 → ⑦ "카드로 추가"
